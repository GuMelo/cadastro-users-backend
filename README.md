# 📦 Cadastro de Usuários - Backend

API backend simples (Estudo de NodeJS) para gerenciamento de usuários (CRUD), permitindo listar, adicionar e remover usuários.

## 🚀 Tecnologias utilizadas

- Node.js
- Express
- (adicione aqui se estiver usando: Prisma, MongoDB, PostgreSQL, etc.)

## 📂 Funcionalidades

- ✅ Listar usuários
- ✅ Criar usuário
- ✅ Deletar usuário
